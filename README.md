# ylc-skills

A Claude Code **plugin marketplace** that includes the `project-learning-report-fastapi` and `secure-code-review` plugins.

## Quick start (local)
```bash
# In Claude Code
/plugin marketplace add ./ylc-skills
/plugin install project-learning-report-fastapi@ylc-skills
/plugin install secure-code-review@ylc-skills
```

Then ask Claude to generate a FastAPI project learning report (using the `/project-learning-report-fastapi` skill) or perform a security code review.
